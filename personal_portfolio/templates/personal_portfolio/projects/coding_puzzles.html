LeetCode is a very popular and well known platform that offers coding challenges online in various languages. Many interviews pull coding questions from LeetCode or similar sites. LeetCode also offers a daily coding challenge and specific questions for interview prep or specific topics of your choosing. I currently focus on doing problems in Python and SQL but plan on branching out and tackling problems in many languages. I may also revisit some problems to optimize them or attempt them in a different language.

Advent of Code is a yearly coding challenge set up as an Advent Calendar. Each day has a new Christmas themed coding challenge consisting of two parts. I decided to participate this year as a way to get more practice creating algorithms for novel problems. All problems can be done in any language of choice. I have opted to use Python for this year. I am currently still working through these challenges but plan on completion before the end of January.
